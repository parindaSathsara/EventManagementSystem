Complete Requirement Coverage

What Is Included in the Project

1. Accounts and Access

  • Phone OTP login (primary), optional email login
  • User roles: Customer, Artist, Admin
  • Account settings and notification preferences

2. Artist Verification System

  • Verification request with ID and social links
  • Admin approve, reject, suspend actions
  • Only verified artists can publish reels and events

3. Social Feed (Reels)

  • Vertical reels feed with smooth autoplay
  • React, repost, save, share, follow actions
  • No public comments for customers

4. Event Management

  • Artist and admin event creation workflows
  • Draft, publish, cancel, reschedule lifecycle
  • Event pages with lineup, date, venue, policy

5. Calendar and Discovery

  • Month view and day list view calendar
  • Near me filtering with location radius
  • Search by artist, event, city, category

6. Notifications

  • Push notifications for events and engagement
  • Major event change alerts (date, venue, cancel)
  • Reminder notifications before event time

7. Moderation and Trust

  • Report flow for reels, events, profiles, repost text
  • Admin moderation queue and actions
  • Auto-hide logic for mass reports

8. Marketing and Growth

  • Deep links for reel, event, artist pages
  • Web preview pages for non-app users
  • UTM attribution tracking from campaigns

9. Admin Dashboard

  • User and role management
  • Artist verification queue
  • Event ops, venue management, reporting exports

Technical Base Included

  • React Native mobile app and React admin web
  • Node.js + TypeScript + NestJS backend
  • PostgreSQL + PostGIS, Redis, object storage, CDN
  • Secure authentication, access control, monitoring

Business Rules Included

  • Event change logs and policy based notifications
  • Age restriction labeling and filtering
  • Privacy controls for repost visibility
  • Analytics tracking events for growth and reporting

Build Approach

Phase Plan (Now and Next)

Phase 1 - Build Now (Included in this proposal)

  • Core social platform and verified artist publishing
  • Event creation and event discovery calendar
  • Admin dashboard and moderation operations
  • Deep links, campaign attribution, analytics events
  • Optional free ticket reservations model

Phase 2 - After Traction (Future extension)

  • Paid ticketing and payment gateway integrations
  • Refund automation, waitlists, and payout management
  • Anti-fraud ticket improvements and offline scanning
  • Advanced finance reporting and AI moderation
